 # 🏡 Airbnb Clone App (TrueStay)

A full-stack Airbnb clone web application where users can create, explore, and review property listings with image uploads and secure authentication.

## 📌 Project Overview

This project is a complete clone of Airbnb-style listing platform built using MERN-like stack (without React). It allows users to browse listings, upload properties, add reviews, and manage their own listings.

---

## ✨ Features

- 🔐 User Authentication (Signup / Login / Logout)
- 🏠 Create, Edit, Delete Listings (CRUD)
- 🖼️ Image Upload using Cloudinary
- ⭐ Review & Rating System
- 🔍 Search functionality
- 📂 Category-based Filtering
- 📱 Fully Responsive UI
- ⚡ Flash Messages & Error Handling
- 🛡️ Secure Sessions using Express-Session
