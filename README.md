# 🏡 TrueStay  

TrueStay is a full-stack house listing platform where users can add, edit, and explore property listings with location maps and reviews.  

---

## ✨ Features  
• Add, edit & delete property listings  
• Authentication & authorization (only owners can edit/delete their listings)  
• Users can add and delete their own reviews  
• Mapbox integration to show property location  
• Search listings by title, place, or country  
• Category filter for quick discovery  
• Responsive design for desktop & mobile  

---

## ⚙️ Tech Stack  
• **Backend:** Node.js, Express.js  
• **Frontend:** EJS, HTML, CSS, JavaScript  
• **Database:** MongoDB Atlas  
• **Auth:** Passport.js  
• **Maps:** Mapbox API  
• **Deployment:** Render  

---

## 🚀 Run Locally  

```bash
git clone https://github.com/Yashin-53/TrueStay-app.git
cd TrueStay-app
npm install
npm start
```
